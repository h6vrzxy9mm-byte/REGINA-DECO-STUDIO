
# Regina Deco Studio

App inicial para catálogo, cotizador y reservas de Regina Deco.

## Funciones iniciales
- Paquetes de decoración
- Cotizador automático
- Selección de paneles
- Cilindros
- Neones
- Globos
- Consulta de disponibilidad
- Panel administrador

## Ejecutar
```bash
pip install -r requirements.txt
streamlit run app.py
```
